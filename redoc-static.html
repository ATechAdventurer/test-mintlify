Device Location

Operations related device locations.

Returns location details about a specified SIM.

Returns location details about a specified SIM.

Request Body schema: application/json

The details of the device location request

iccid
string (Iccid)

The SIM card's ICCID.

horizontalPrecision
string
Value: "AREA"

The horizontal precision of the location.

respTimeOut
integer [ 0 .. 50 ]

The response timeout in seconds.

maxCacheAge
integer [ 0 .. 900 ]

The maximum cache age, in seconds.

By comparing the cache with maxCacheAge you can tell whether the information was retrieved from the server or from the cache:

cache > maxCacheAge: Information is sent from the server cache < maxCacheAge: Information is sent from the cache

Responses

https://naas.t-mobile.com/iot-location/v1/locationCheck
/{apiRoot}/{basePath}/iot-location/v1/locationCheck
